# Eye Movement Tracking Dashboard

A real-time eye movement tracking dashboard built with MediaPipe, OpenCV, ipywidgets, and Plotly. This dashboard captures webcam video, processes eye movements, and displays various metrics and visualizations.

## Features

- **Real-Time Video Feed:** Displays the live video with facial landmarks.
- **Metrics Display:** Shows 13 eye movement metrics including saccade speed, blink count, fixation duration, pupil diameter, and more.
- **Interactive Graphs:** Visualizes key metrics over time using Plotly.
- **Data Recording:** Saves all metrics with timestamps to a CSV file upon stopping the stream.
